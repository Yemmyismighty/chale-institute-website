### Metadata

## Image Guidelines

- Use high-quality images (minimum 1200px width for hero/featured images)
- Optimize images before adding them (use tools like ImageOptim, TinyPNG)
- Use descriptive filenames
- Ensure images are properly licensed for use
- Consider using WebP format for better performance
- Next.js Image component will automatically optimize images at build time
